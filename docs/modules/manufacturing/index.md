# Manufacturing

Manufacturing is where raw materials become finished goods — and where Nama ERP helps you plan that transformation, record what actually happened on the shop floor, and put a real cost on every unit you produce. The module walks the full production cycle: deciding what to make, making sure the materials are there, executing the work, and then following the money so your product costs are accurate.

If you plan production, run a shop floor, manage material requirements, or need to understand product costing, start with the overview and follow the cycle through orders, execution and costing. There's also a dedicated set of guides for carton and corrugated-product manufacturers, whose material planning has its own optimization engine.
