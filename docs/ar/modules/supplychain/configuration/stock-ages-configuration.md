# إعدادات أعمار المخزون

تشرح هذه الصفحة تبويب **أعمار المخزون**. خاصية أعمار المخزون هي خاصية تتبّع تقادم المخزون: فهي تتابع المدة التي ظل فيها المخزون محتفظاً به حتى تتمكن من إعداد تقارير عن شرائح التقادم. تُبنى سجلات التقادم بواسطة معالجة تعمل في الخلفية، وتحدد هذه الإعدادات ما إذا كان التقادم يُتابَع أصلاً، وإلى أي مدى يُفصَّل المخزون المحتفظ به.

**متابعة أعمار المخزون** *(`value.stockAgesPolicy`)* — المفتاح الرئيسي لتتبّع التقادم، ويُحَل من خلال سلسلة وراثة (الصنف ← مجموعة الصنف ← هذه الإعدادات)، بحيث يمكن للصنف أن يتبع السياسة العامة أو يتجاوزها. اضبطه على **نعم** لتفعيل تقادم المخزون، أو **لا** لتعطيله، أو **موروث** في المستويات الأدنى لاتباع الإعداد العام.

## محددات المتابعة

كل مفتاح أدناه يحدد ما إذا كان ذلك المحدد سيُؤخَذ في الاعتبار عند بناء سجلات التقادم. عندما يكون المحدد **مغلقاً**، يُجمَّع المخزون المحتفظ به بغض النظر عنه؛ وعندما يكون **مفتوحاً**، يُتابَع التقادم بشكل منفصل لكل قيمة مختلفة من قيم ذلك المحدد — تفصيل أدق، لكن سجلات أكثر للبناء والتخزين. فعّل المحددات التي تحتاج إلى متابعة تقادم المخزون عليها، واترك الباقي مغلقاً لإبقاء بيانات التقادم أصغر وأسرع.

**متابعة أعمار المخزون على:**

| المحدد | معرّف الحقل |
|---|---|
| القطاع | `value.trackStockAgesOnSector` |
| الفرع | `value.trackStockAgesOnBranch` |
| الإدارة | `value.trackStockAgesOnDepartment` |
| المجموعة التحليلية | `value.trackStockAgesOnAnalysisSet` |
| المخزن | `value.trackStockAgesOnWarehouse` |
| الموقع | `value.trackStockAgesOnLocator` |
| الإصدار | `value.trackStockAgesOnRevision` |
| المقاس | `value.trackStockAgesOnSize` |
| اللون | `value.trackStockAgesOnColor` |
| التعبئة (الصناديق) | `value.trackStockAgesOnPackages` |
| الشحنات | `value.trackStockAgesOnLotId` |
| النسبة الفعالة | `value.trackStockAgesOnActivePercent` |
| النسبة غير الفعالة | `value.trackStockAgesOnInactivePercent` |
| الصنف الفرعي | `value.trackStockAgesOnSubItem` |

## المعالجة

**أوقات معالجة أعمار المخزون** *(`value.stockAgesProcessingTimes`، جدول)* — النوافذ الزمنية (بداية/توقف) التي يُسمَح خلالها لمعالجة أعمار المخزون التي تعمل في الخلفية بالتشغيل، حتى يمكن حصر عمل التقادم الثقيل في ساعات انخفاض الضغط.
