# Entity Flows

Entity Flows let you automate the system's reaction to everyday actions — saving, editing, revising a record — without writing any code. They put capabilities that used to belong to developers into the hands of consultants and support staff: generate one entity from another, import data straight from Excel or a SQL query, and much more. Start with the introduction, then work through the examples and FAQ as you build your own.
