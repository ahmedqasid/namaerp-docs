# Business Intelligence

The BI module turns your live ERP data into interactive dashboards — charts, gauges, tables, and cards that talk to each other through cross-filters, drill-downs, and direct links. Start with the guide to learn what you can build by clicking around, then dip into the technical references when you need the exact JSON, SQL, and widget options behind a particular feature.
