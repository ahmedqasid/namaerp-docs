# Human Resources

The Human Resources module is where Nama ERP keeps track of your people and pays them correctly — from the raw fingerprint records a time-attendance machine spits out, all the way to a finished payroll run. The guides here focus on the part that trips teams up most: getting attendance data into the system cleanly, so that lateness, overtime and absences feed the right way into salaries.

If you import time-attendance files, configure attendance and departure rules, or run payroll, these pages are for you. They explain how Nama reads the many different file formats machines produce, how it handles overlapping or duplicate punch lines, and answer the questions that come up most around payroll.
