# Notifications

When something important happens in the system — a new invoice, an approval request, a processing error — the right person should hear about it without anyone remembering to tell them. Nama's notification system watches for business events and automatically sends messages through in-app alerts, email, SMS, and WhatsApp. Learn how the engine and templates work, wire up your SMS and WhatsApp providers, and check the FAQ when a message doesn't go where you expected.
