# قوائم العرض

قوائم العرض هي الشاشات التي تتصفّح فيها سجلاتك وتبحث عنها. وثمة لمستان بسيطتان تجعلان التعامل معها أسرع بكثير يوميًا: الفلاتر السريعة التي تعثر على السجلات التي تريدها بنقرة واحدة، وإعداد يُبقي المستخدم داخل سنة مالية واحدة. يغطّي هذان الدليلان القصيران كليهما.
