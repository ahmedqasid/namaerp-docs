# Approvals

Some documents shouldn't go through without a sign-off — a large purchase order, a stock transfer, a discount above a threshold. Nama's approvals system lets you route any record through one or more approval steps, decide who approves at each step, and notify them by email, SMS, or in-app message. Start with the full definition guide, grab a ready-made email template, and check the FAQ for real configuration recipes.
